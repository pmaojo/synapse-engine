# Research Assistant Scenario

This scenario is designed to help you organize academic research.

## Concepts
* **Paper**: A scientific publication.
* **Author**: A researcher who wrote a paper.
* **authoredBy**: The relationship between a paper and its author.

## Usage
When processing a paper, identify the title, authors, and year.
Create `Paper` nodes for each publication.
Create `Author` nodes for each person.
Link them using `authoredBy`.
