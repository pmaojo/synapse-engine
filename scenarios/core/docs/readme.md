# Synapse Core Scenarios

This scenario provides the fundamental ontologies for the Synapse Semantic Engine.

## Included Ontologies
* **Schema.org**: Person, Organization, Action, Event.
* **PROV-O**: Provenance tracking (wasDerivedFrom, generatedAtTime).
* **SKOS**: Concepts and schemes.
* **FOAF**: Friend of a Friend (relationships).
* **Memory**: Specialized ontology for episodic memory.
